# n1mods
These are mods for the browser game https://narrow.one . Run them in tampermonkey.

**MODS**

Mod_1: Simple modpack which adds some cool and helpful features to the game.

Mod_2: 1v1 mod, which adds a scoreboard to the UI. Only works with exactly 2 people in the round.

Mod_3: Hack, which adds cheat-features such as high jump, speed, godmode, xray and more.


Legacy: Mod_1 and Mod_2 are confirmed and allowed by the developers of narrow one. Mod_3 is a hack. You are only allowed to use it in private rounds and you might get banned for using it in public. Mod_3 will ask you to enter a password to make sure, you are ok with the risks. Password: 45690
